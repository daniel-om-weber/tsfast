Corefunctions

Corefunctionality for creating dataloaders objects from downloaded datasets

source

create_dls_from_spec

 create_dls_from_spec (spec:identibench.benchmark.BenchmarkSpecBase,
                       dataset=None, download_function=None,
                       win_sz:int=100, x:list=[], stp_sz:int=1,
                       sub_seq_len:int=None, bs:int=64,
                       prediction:bool=False, input_delay:bool=False,
                       valid_stp_sz:int=None, cached:bool=True,
                       num_workers:int=5, max_batches_training:int=1000,
                       max_batches_valid:int=None, dls_id:str=None,
                       **kwargs)

Create a dataloaders object from identibench benchmark specification. Extracts benchmark specific kwargs from BENCHMARK_DL_KWARGS and adds them to the kwargs for create_dls_downl.

Type Default Details
spec BenchmarkSpecBase Specification of the benchmark from identibench
dataset NoneType None
download_function NoneType None
win_sz int 100
x list []
stp_sz int 1
sub_seq_len int None
bs int 64

input_delay bool False
